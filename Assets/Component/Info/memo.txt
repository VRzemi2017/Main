・使用するプレハブとスクリプトについて
	各フォルダ内のInfo_True内にあるものを使ってください。

・シーンについて
	Info_TitleとInfo_Mainがメインです。

・infomation呼び出し時

	呼び出したいタイミングで
		InfoManager.CallCase(InfoManager.InfoCase.INFO_〇〇);を追加
			→InfoManager.cs内のInfoCaseにパターンの追加、
			　Update内で表示する文字の決定
				※同時に別のinfomationの表示が現状不可能

・Infomationの呼び出しテストについて
	どのキーを使っているのかわからなくなるので、
	infoManager内のUpdateで
	if (Info_Case == InfoCase.INFO_TEST)
        {
			[呼び出したいウィンドウ]
        }

	InfoCall_testのUpdateで呼び出しをお願いします。
	追記…ヒエラルキー上のInfoTesterにInfoCall_testをくっつけてあるので、
			使わない場合はオブジェクトを消してください

・プレイヤーの名前表示について
	各カメラか何かの上に文字を表示して、"Player_Icon"内のスクリプトを張り付けてください。
